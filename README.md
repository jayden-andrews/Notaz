# Notaz

## Elevator Pitch  
Can you read sheet music? If not, you’re not alone. **Notaz** is a full-stack learning app designed for beginners, using interactive exercises to help users quickly build confidence and fluency in reading music.

---

## Problem Statement  
The ability to read sheet music is becoming less common, especially among self-taught musicians. Existing tools can feel overwhelming or outdated, so **Notaz** makes learning approachable through simple, structured, and engaging practice.

---

## Target User  
A beginner musician who wants to learn how to read sheet music but feels intimidated by traditional methods. They’re motivated to improve but need something interactive, flexible, and easy to stick with.

---

## Why This Project  
Music has been a major part of my life for over 14 years, shaping both my skills and opportunities. I want to make music literacy more accessible so others can experience the same growth, confidence, and creative freedom that music has given me.
